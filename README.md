#Intelligent Classroom Attention Monitoring System
## Description :
This project is an Object-Oriented Programming implementation of a classroom attention monitoring system.
It demonstrates concepts like Inheritance, Polymorphism, Encapsulation, Composition and Dynamic memory.

## Features :
- Add Students
- Simulated Emotion Detection
- Attention Level Calculation
- Class Average Analysis
- Menu-driven Interface

## Technologies Used
- C++
- Object-Oriented Programming Concepts
